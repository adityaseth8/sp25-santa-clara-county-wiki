== Overview ==
A cloud computing platform offered by Microsoft which provides on demand delivery of IT resources such as networks, servers, computing, and storage solutions.

== How it is Used ==
The County uses Azure for a wide variety of operations, such as hosting public-facing County websites, storing databases of various County departments, setting security controls for data access, and more.
